# Primeiros Passos

Acesse o Suri Tools pelo link interno fornecido pelo time de tecnologia. A plataforma pode ser instalada como um **Progressive Web App (PWA)** para acesso offline e experiência nativa.

> [!TIP]
> Clique em **"Instalar App"** no dashboard para adicionar o Suri Tools à tela inicial do seu dispositivo.

## Requisitos

1. Navegador moderno (Chrome 90+, Safari 15+, Edge 90+)
2. Conexão à rede corporativa para funcionalidades de API
3. Credenciais de acesso ao sistema TOTVS

## Suporte

Em caso de dúvidas ou problemas, entre em contato com o time de tecnologia via canal interno.
