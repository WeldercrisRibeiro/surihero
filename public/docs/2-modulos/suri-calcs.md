# Suri Calcs

O **Suri Calcs** foi desenvolvido para agilizar o cálculo de propostas de **UpSell** e **DownSell** durante atendimentos comerciais.

## Funcionalidades

- Cálculo automático de totais por linha de produto
- Geração de orçamento com um clique
- Validação de valores zerados para evitar propostas inválidas

> [!CAUTION]
> O botão **"Gerar Orçamento"** fica desabilitado quando o valor total é zero. Certifique-se de preencher ao menos um item antes de gerar.

## Como calcular um UpSell

1. Selecione o contrato atual do cliente.
2. Adicione os módulos ou usuários a incluir.
3. O valor diferencial será calculado automaticamente.
4. Clique em **Gerar Orçamento** para exportar.

## Como calcular um DownSell

1. Selecione o contrato atual do cliente.
2. Remova os módulos ou usuários a excluir.
3. Revise o valor de crédito gerado.
4. Confirme e clique em **Gerar Orçamento**.
