# Kanban

O módulo **Kanban** oferece um quadro visual de tarefas com colunas configuráveis, arrastar-e-soltar e filtros por tag.

## Operações básicas

- Adicione colunas com o botão **+ Coluna**
- Crie cartões clicando em **+ Cartão** dentro de uma coluna
- Arraste cartões entre colunas para mudar o status
- Clique em um cartão para editar detalhes

## Gerenciando colunas

1. Clique no ícone de lápis ao lado do nome da coluna para renomeá-la.
2. Use o ícone de arrastar (⠿) no cabeçalho para reordenar colunas.
3. Clique no ícone de lixeira para excluir uma coluna vazia.

> [!WARNING]
> Excluir uma coluna com cartões remove todos os cartões dentro dela permanentemente.

## Filtros e busca

Use a barra de busca no topo para filtrar cartões por título ou descrição em tempo real.
