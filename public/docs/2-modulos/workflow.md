# WorkFlow

O **WorkFlow** é um editor visual para mapear processos e fluxos de trabalho usando nós conectados por setas.

## Dicas de uso

- Dê **duplo clique** em um nó para renomeá-lo
- Clique em uma **seta** para estilizá-la (cor e pontilhado)
- Use o botão **Exportar** para salvar o fluxo como imagem PNG
- O título do fluxo pode ser editado diretamente na topbar com duplo clique

## Salvando fluxos

1. Clique em **Salvar** na topbar.
2. Preencha o título e a descrição opcional.
3. O fluxo aparecerá no painel lateral direito.

> [!NOTE]
> Os fluxos são salvos localmente na memória da sessão atual. Ao recarregar a página, eles são perdidos. Exporte como PNG para guardar permanentemente.

## Atalhos de teclado

- `Backspace` ou `Delete` – Remove o nó ou seta selecionado
- `Ctrl + Scroll` – Zoom in/out no canvas
