# Bem-vindo ao Suri Tools

O **Suri Tools** é uma plataforma interna criada para centralizar ferramentas do time. Aqui você encontra módulos de API, calculadoras, fluxos de processo e muito mais.

> [!NOTE]
> Esta documentação cobre a versão **2.1.0** da plataforma. Versões anteriores podem ter comportamentos diferentes.

## Módulos disponíveis

- **Suri API** – Teste e valide endpoints da API TOTVS diretamente do navegador.
- **Suri Calcs** – Calculadora de UpSell e DownSell com geração de orçamento.
- **Kanban** – Quadro visual para gestão de tarefas da equipe.
- **Flows** – Editor de fluxos visuais para mapeamento de processos.
- **Docs** – Este módulo de documentação centralizada.

## Como navegar

Use o painel lateral esquerdo para navegar pelas categorias e documentos. Você pode usar a busca rápida para encontrar um tópico específico.

O botão **Ocultar / Sumário** na topbar alterna a visibilidade da sidebar.
